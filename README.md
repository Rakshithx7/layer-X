# 🚀 Layer X

### *Humanizing Blockchain Interactions on Solana*

---

## 🧠 Overview

**Layer X** is a next-generation interaction layer for Solana that transforms complex blockchain operations into simple, human-readable commands.

Instead of navigating confusing wallets, copying long addresses, or interpreting raw transaction logs, users can simply type what they want, just like sending a message.

> 💬 *"Send 1 SOL to @pooja"*  
> ⚡ → Executed on-chain

---

Despite the rapid growth of Web3, **user experience remains a major barrier**.

### ❌ Current Challenges

- Long and error-prone wallet addresses

- Lack of human-readable interfaces

- Fragmented DeFi interactions

> **Crypto is powerful, but not human-friendly.**

---

## 💡 Our Solution

Layer X introduces a **Natural Language Interaction Layer (NLIL)** that abstracts blockchain complexity into intuitive commands.

We bridge the gap between **human intent** and **on-chain execution**.

---

## ⚙️ How It Works

```text
User Input → Parser → Structured Action → Identity Resolution → Transaction Builder → Wallet Sign → On-chain Execution
```

### Flow

1. User types a command
2. Parser extracts intent (action, amount, token, recipient)
3. Identity layer resolves `@username` → wallet address
4. Transaction preview is generated
5. User signs via wallet (Phantom)
6. Transaction is executed on Solana

## 🔥 Core Features

### 💬 Chat-Based Transactions

Execute blockchain actions using natural language:

```text
send 1 SOL to @pooja
swap 1 SOL to USDC
```

### 👤 Identity Layer (@username)

- Map human-readable names to wallet addresses
- Eliminate copy-paste errors
- Personalized crypto experience

### 🔐 Safe Transaction Preview

Before signing, users see:

- Amount
- Recipient
- Fees
- Warnings

### 🧠 Smart Explorer (Human-readable)

Convert raw blockchain data into simple summaries:

```text
Instead of:
Instruction logs, accounts...

We show:
→ Sent 1 SOL to @pooja
```

### ⚠️ Risk Detection Layer

Prevent user mistakes with intelligent warnings:

- New address detection
- Large transaction alerts
- Unverified tokens
- Suspicious interactions

### 🔄 DeFi Swap Layer (Aggregator Ready)

- Integrated with Jupiter (mainnet-ready)
- Shows route, price impact, and slippage
- Explains swaps before execution

### 🪙 Token Launchpad (Metaplex Powered)

Create tokens in seconds:

- Name
- Symbol
- Supply
- Logo (IPFS)
- Metadata

> Simplified alternative to complex tools like pump.fun

### 🖼 NFT Support

- Detect NFTs in wallet
- Display metadata (image, name)
- Transfer NFTs via commands

### 📊 Portfolio Overview

- SOL + SPL tokens
- Metadata-aware display
- Clean and minimal UI

## 📦 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install

```bash
npm install
```

### Run in Development

```bash
npm run dev
```

This starts:

- Vite client (`dev:client`)
- Contacts API server (`dev:server`)

### Build for Production

```bash
npm run build
```

### Lint

```bash
npm run lint
```
